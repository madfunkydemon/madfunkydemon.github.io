<p>To re-run the package group selection from the Ubuntu server installation use:</p>
<pre>
sudo tasksel

</pre>
<p>If you want to view the avalable tasks use:</p>
<pre>
sudo tasksel --list-tasks

</pre>
<p>Just make sure you have the install CD in.</p>
